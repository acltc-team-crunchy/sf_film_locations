# SfFilmLocations

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sf_film_locations'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sf_film_locations

## Usage

Get all locations:
  
    $ SfFilmLocations::Location.all
